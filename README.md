## tailwind.config.js
- reserved name config file
- content: configure the paths to all of your HTML templates, JavaScript components, and any other source files that contain Tailwind class names.
- theme: define your project’s color palette, type scale, fonts, breakpoints, border radius values, and more.
- plugins: register new styles for Tailwind to inject into the user’s stylesheet using JavaScript instead of CSS.
